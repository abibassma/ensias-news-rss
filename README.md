# ENSIAS News RSS

Auto-generated RSS feed for the ENSIAS news page: https://ensias.um5.ac.ma/news/

The repository runs a small Python scraper on GitHub Actions that generates `docs/rss.xml` (RSS 2.0) and publishes it via GitHub Pages.
